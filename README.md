# ヘルスケアダッシュボード（health_manager）

Apple Health などからエクスポートした **ZIP 形式のヘルスデータ** をブラウザで読み込み、**日ごとの指標**をグラフで見たり、**Google Gemini** で簡単な分析コメントを得られる、個人用のダッシュボードです。

医療機器や診断ツールではありません。あくまで自分のデータを整理・閲覧するための補助です。

---

## できること

- ZIP をアップロードして、血圧・心拍・カロリー・睡眠・歩数などを **期間指定**して表示
- **トレンド**（折れ線・棒グラフ）で推移を確認
- （API キー設定時）**AI 分析**で集計に基づいた短いコメントを生成
- 分析の口調は `prompts/analysis_system.txt` のテキストで変更可能

## セットアップ（初めての方）

### 1. リポジトリを手元に用意する

#### GitHub からクローンする

[Git](https://git-scm.com/) をインストールしたうえで、ターミナルで次を実行します。

```bash
git clone https://github.com/geekabs/health_manager.git
cd health_manager
```

これで `health_manager` フォルダにソースが入り、**そのフォルダがプロジェクトのルート**になります。以降の手順はこのまま続けてください。

#### すでにフォルダがある場合

別の方法で入手した `health_manager` フォルダがある場合は、そこへ移動します。

```bash
cd /path/to/health_manager
```

（`/path/to/` は実際のパスに読み替えてください。）

### 2. 仮想環境をつくる（おすすめ）

他の Python アプリとパッケージが混ざらないようにします。

**Mac / Linux:**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

**Windows（コマンドプロンプト）:**

```cmd
python -m venv .venv
.venv\Scripts\activate.bat
```

プロンプトの先頭に `(.venv)` のように表示されれば成功です。

### 3. 必要なライブラリを入れる（`health_manager` 内で実行）

```bash
pip install -r requirements.txt
```

### 4. 環境変数ファイルをつくる（同じくプロジェクトルートで）

`.env.example` をコピーして `.env` という名前のファイルを作ります。

**Mac / Linux:**

```bash
cp .env.example .env
```

**Windows（PowerShell）:**

```powershell
copy .env.example .env
```

`.env` をメモ帳やエディタで開き、`GEMINI_API_KEY=` の右側に API キーを貼り付けます。AI 分析を使わない場合は空のままでもダッシュボードの閲覧はできます。

`.env` は **Git に載せない**でください（このリポジトリの `.gitignore` で除外済みです）。

---

## 起動方法

仮想環境を有効にした状態で、次を実行します。

```bash
streamlit run app.py
```

ブラウザが開き、`http://localhost:8501` あたりにダッシュボードが表示されます。止めるときはターミナルで `Ctrl + C` です。

---

## 使い方の流れ

1. サイドバーから **ヘルスデータの ZIP** を選ぶ（ドラッグ＆ドロップでも可）
2. **表示する期間**を「過去7日」などにするか、「カスタム」で開始日・終了日を指定する
3. メイン画面で KPI とグラフを確認する
4. AI 分析を使う場合は `.env` に API キーを入れてから **分析を実行** を押す

分析の文章のルールを変えたいときは、`prompts/analysis_system.txt` を編集して保存し、再度分析を実行してください。

## トラブル時

- **`streamlit` が見つからない**  
  → 仮想環境を有効にしてから `pip install -r requirements.txt` をもう一度。
- **ZIP が読めない**  
  → Apple Health の公式エクスポート形式か、対応している CSV を含む ZIP か確認してください。
- **AI 分析だけ動かない**  
  → `.env` の `GEMINI_API_KEY` と、インターネット接続を確認してください。
